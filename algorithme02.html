Pour faire la division entière on utilise div pour obtenir le quotient et mod pour obtenir le reste. 

pommes ← 26
// Calculer le nombre de groupes de sept pommes
nbg ← pommes div 7 // nbg = 26 div 7 = 3
// Calculer le nombre de pommes restantes
npr ← pommes mod 7 // npr = 26 mod 7 = 5
